Getting Started


Installation

For Maven, add dependencies in pom.xml:

<dependency>
  <groupId>ai.djl.tensorflow</groupId>
  <artifactId>tensorflow-engine</artifactId>
  <version>0.x.x</version>
</dependency>

How to Download and Set Up DJL

Step 1: Prerequisites

Ensure that you have the following installed on your machine:

  • Java Development Kit (JDK): DJL requires Java 8 or higher.
  • Maven or Gradle: These are build tools for managing dependencies in your Java projects.

Step 2: Create a New Java Project

Using Maven:

mvn archetype:generate -DgroupId=com.example -DartifactId=djl-example -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
cd djl-example

Using Gradle:

mkdir djl-example
cd djl-example
gradle init --type java-application

Step 3: Add DJL Dependencies

For Maven

<dependencies>
  <dependency>
    <groupId>ai.djl.tensorflow</groupId>
    <artifactId>tensorflow-engine</artifactId>
    <version>0.x.x</version> <!-- Replace with the latest version -->
  </dependency>
  <dependency>
    <groupId>ai.djl.tensorflow</groupId>
    <artifactId>tensorflow-native</artifactId>
    <version>0.x.x</version> <!-- Replace with the latest version -->
  </dependency>
  <dependency>
    <groupId>ai.djl.pytorch</groupId>
    <artifactId>pytorch-engine</artifactId>
    <version>0.x.x</version> <!-- Replace with the latest version -->
  </dependency>
</dependencies>

For Gradle

dependencies {
  implementation 'ai.djl.tensorflow:tensorflow-engine:0.x.x' // Replace with the latest version
  implementation 'ai.djl.tensorflow:tensorflow-native:0.x.x' // Replace with the latest version
  implementation 'ai.djl.pytorch:pytorch-engine:0.x.x' // Replace with the latest version
}

Step 4: Build the Project

After adding the dependencies, build your project:

mvn clean install // For Maven
gradle build // For Gradle

Step 5: Verify the Setup

Create a simple Java class to verify that DJL is set up correctly:

import ai.djl.Model;
import ai.djl.translate.TranslateException;

public class DJLTest {
  public static void main(String[] args) {
    try (Model model = Model.newInstance("model-path")) {
      System.out.println("DJL is set up correctly!");
    } catch (TranslateException e) {
      e.printStackTrace();
    }
}
}

Step 6: Run Your Application

In your IDE, run the DJLTest class. If everything is set up correctly, it should run without errors.
